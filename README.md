Nombre del proyecto: Inventario de Consumibles.

El proyecto fue creado en Excel con Macros y Visual Basic 6.0 integrado en Excel, permite llevar el control de un pequeño inventario, registrando productos nuevos en la base de datos, realizando ingresos al inventario, y salidas que serían los consumos, tambien trazabilidad de los movimientos realizados.

Características

- Impulsó el control del inventario.
- Deja huella de los movimientos de las materias primas en el inventario.
- Mejor rotación de las referencias en el inventario.

Requisitos Previos

Antes de comenzar, asegúrate de tener:

- Microsoft Excel.
- Que las macros se puedan ejecutar en tu PC.
- El archivo no tiene contraseñas, por lo que es totalmente modificable.
